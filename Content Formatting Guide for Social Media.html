Content Formatting Guide

Single Image Post

Perfect for infographics or photos.

  • Size: 1200x675px (1.91:1) or 1080x1350px (4:5).
  • Formats: PNG (infographics), JPEG (photos), max 5 MB.
  • Tips: Use alt text, compress with TinyPNG.

Carousel Post

Multiple images for storytelling.

  • Size: 1200x675px (1.91:1) or 800x418px (X, 7:8).
  • Formats: PNG/JPEG, max 5 MB per image.
  • Tips: Consistent aspect ratios, number images.

Video Post

Promotional or tutorial videos.

  • Size: 1280x720px (16:9) or 1200x1200px (1:1), max 512 MB.
  • Formats: MP4/MOV (H.264, AAC audio).
  • Tips: Keep <2m20s for X, use a compelling thumbnail.

Text with Image Post

Text-heavy post with an inline image (e.g., a quote).

  • Size: 1200x675px (1.91:1), max 5 MB.
  • Formats: PNG/JPEG.
  • Tips: Use bold typography, high-contrast text.

Article Post

Link to a blog/article with a preview image.

  • Size: 1200x675px for og:image, max 5 MB.
  • Formats: PNG/JPEG.
  • Tips: Use OG tags (title <90, desc <160 chars), test with validators.

X (Twitter)

Format content for X to maximize engagement.

  • Image Sizes: 1200x675px (1.91:1), 800x800px (1:1), GIF 1280x1080px.
  • Formats: JPEG, PNG, WebP, GIF. No TIFF/BMP. Max 15 MB on mobile.
  • OG/Twitter Tags: `og:image` (1200x675px), `twitter:card` ("summary_large_image").
  • Post Length: 280 chars (free), 25,000 chars (premium). URLs = 23 chars.
  • Tips: Center key info, test with the X Card Validator.

LinkedIn

Professional previews for LinkedIn.

  • Image Sizes: 1200x627px (1.91:1), 1200x1200px (1:1), max 5 MB.
  • Formats: JPEG, PNG. No WebP support.
  • OG Tags: `og:image` (1200x627px), `og:title` (<90 chars), `og:description` (<160 chars).
  • Tips: Use PNG for text-heavy images, test with the LinkedIn Post Inspector.

Facebook

Engage audiences on Facebook.

  • Image Sizes: 1200x630px (1.91:1), min 600x315px, max 5 MB.
  • Formats: JPEG, PNG.
  • OG Tags: `og:image` (1200x630px), `og:title` (<90 chars), `og:description` (<160 chars).
  • Tips: Center visuals, test with the Facebook Sharing Debugger.

YouTube

Clickable thumbnails for YouTube videos.

  • Image Size: 1280x720px (16:9), max 2 MB.
  • Formats: JPEG, PNG, GIF, BMP.
  • Safe Zone: Keep text/logos away from the bottom right corner (timestamp overlay).
  • Tips: Use bold text, high-contrast colors, and expressive faces to increase CTR.
